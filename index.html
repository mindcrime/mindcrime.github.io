Fogbeam

As Founder / CEO at Fogbeam Labs I contribute code to all of our various open source projects, including:
  • Quoddy - Enterprise Social Network
  • Neddick - Information Discovery Platform(think "Reddit for Organizations")
  • Heceta - Enterprise Search Engine
  • Hatteras - Business Events Subscription Engine

In addition to our own projects, we integrate with, extend, support, or otherwise use, a number of other open source projects. We have two demo servers running where you can poke and prod at both our projects and some of the other cool stuff we work with.

  • demo.fogbeam.org - Quoddy and Neddick
  • demo2.fogbeam.org - Stanbol, OpenMeetings, Zeppelin

Other Projects

  • ArrowSmith - The beginnings of an open source implementation of ArrowSmith using Groovy and Grails. This is meant to be a testbed for experimenting with Literature Based Discovery.
  • OpenNLP Demo - some demo code from a talk on Apache OpenNLP that I presnted at Tri-JUG a while back.
